## Tech Stack and Requirements

-   Next.js
-   Typescript
-   Tailwind css
-   DaisyUI
-   Eslint
-   Prettier

## How to run

1.  clone
2.  cd project
3.  yarn install
4.  set vscode default formatter to ESLint
5.  `npm run dev`
6.  Happy developing!
